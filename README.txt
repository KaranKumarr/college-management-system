//Follow these instructions carefully to run the web application

//Pre-requisites - You need these things to run the application

->NodeJS
->Browser
->Xampp(or anyother server that can live the sql file provided in /Database folder)

Step 1) Import the college-management-system.sql file into your XAMPP server
Step 2) Start the xampp server
Step 3) Open the cmd in application directory and type this in terminal "pm install" (essential for first try)
Step 4) Now finally run this command in terminal "npm start" and check localhost:3000
